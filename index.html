Mastodon Share

Toegankelijk en meertalig deel-via-Mastodon-knopje dat je Mastodon-server onthoudt.

Basisgebruik

Voeg een link toe met de classname mastodon-share en initialiseer het script:

Deel via Mastodon 
<a class="mastodon-share" href="https://share.joinmastodon.org/">
  Deel via Mastodon
</a>

<script type="module">
  import initMastodonShare from 'mastodon-share.js';
  initMastodonShare();
</script>

Bij het eerste klikken verschijnt een dialoog om je Mastodon-server in te voeren. Na opslaan wordt de link bijgewerkt en de server onthouden in localStorage.

Meerdere knoppen

Alle elementen met de classname mastodon-share op de pagina worden in één keer geïnitialiseerd. Ze delen dezelfde opgeslagen server en één dialoogvenster.

Deel via Mastodon Deel via Mastodon 
<!-- Koptekst -->
<a class="mastodon-share" href="https://share.joinmastodon.org/">Deel via Mastodon</a>

<!-- Voettekst -->
<a class="mastodon-share" href="https://share.joinmastodon.org/">Deel via Mastodon</a>

<!-- Één aanroep initialiseert alle knoppen -->
<script type="module">
  import initMastodonShare from 'mastodon-share.js';
  initMastodonShare();
</script>

Aanpassen

Opties worden meegegeven als object, via een globale variabele vóór het laden van de module, of direct als argument:

<!-- Via globale variabele -->
<script>
  var mastodon_sharebutton = {
    dialog_save: "Opslaan en delen",
    via: "@mijnaccount@mastodon.social"
  };
</script>
<script type="module">
  import initMastodonShare from 'mastodon-share.js';
  initMastodonShare(mastodon_sharebutton);
</script>

<!-- Of direct als argument -->
<script type="module">
  import initMastodonShare from 'mastodon-share.js';
  initMastodonShare({
    dialog_save: "Opslaan en delen",
    via: "@mijnaccount@mastodon.social"
  });
</script>

Opties

Optie Standaard Omschrijving
anchor_class "mastodon-share" CSS-classname waarmee de deellinks worden gevonden.
hidden_class "visuallyhidden" Klasse voor de visueel verborgen toegankelijkheidstekst. Vereist eigen CSS (zie Stijlen).
storage_key "mastodon_sharebutton_key" localStorage-sleutel voor het opslaan van de serverinstantie.
linktext_share_initialized "Deel via Mastodon" Toegankelijke linktekst wanneer er een server is opgeslagen.
linktext_share_uninitialized (instructietekst) Toegankelijke linktekst wanneer er nog geen server is ingesteld.
edit_instance "Bewerk je Mastodon-server" Label en aria-label voor de bewerk-knop.
dialog_label "Wat is je Mastodon server?" Koptekst van het dialoogvenster.
dialog_hint (voorbeeldtekst) Helptekst bij het invoerveld in de dialoog.
dialog_hint_invalid (foutmelding) Foutmelding bij ongeldige invoer. Gebruik %SERVER% als plaatshouder voor de ingevoerde waarde.
dialog_cancel "Sluiten" Label voor de sluitknop van de dialoog.
dialog_save "Ga verder" Label voor de bevestigknop van de dialoog.
via "@DigitaleOverheid@…" Optionele vermelding toegevoegd aan de gedeelde tekst. Stel in op "" om te verwijderen.

Stijlen

Het script voegt onderstaande classnamen en elementen toe aan de DOM. Alle visuele opmaak schrijf je zelf.

Selector Onderdeel Omschrijving
.mastodon-share Deellink De originele ankerlink (instelbaar via anchor_class).
.mastodon-share__link Geïnitialiseerde link Class die na initialisatie aan de deellink wordt toegevoegd.
.mastodon-share__link-icon Icoon Span met het Mastodon-SVG-icoon, voor de linktekst geplaatst.
.mastodon-share__link__text Toegankelijke tekst Visueel verborgen span met de beschrijvende linktekst. Krijgt ook de hidden_class (standaard .visuallyhidden).
.mastodon-edit Bewerk-knop Knop die verschijnt nadat een server is opgeslagen. Staat direct na de deellink in de DOM.
.mastodon-edit__text Bewerk-knoptekst Span met de zichtbare tekst van de bewerk-knop.
.mastodon-share-dialog Dialoog Eén gedeeld dialoogvenster, als laatste child van <body> toegevoegd.
.mastodon-share-dialog__hint Dialooghint Helptekst bij het invoerveld; bevat de foutmelding bij ongeldige invoer.
.mastodon-share-dialog.is-invalid Dialoog (foutstate) State-class op de dialoog bij ongeldige serverinvoer.
